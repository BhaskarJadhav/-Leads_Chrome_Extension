# LinkedIn Leads Tracker

A lightweight Chrome extension for saving useful links while researching prospects, companies, or opportunities.

## Features

- Save a URL entered manually
- Save the active browser tab
- Keep saved links in browser local storage
- Open saved links in a new tab
- Clear the complete saved list

## Install

1. Clone or download this repository.
2. Open `chrome://extensions` in Chrome.
3. Enable **Developer mode**.
4. Select **Load unpacked**.
5. Choose the repository folder.

The extension is then available from the Chrome toolbar.

## Built with

`JavaScript` `HTML` `CSS` `Chrome Extensions Manifest V3`

## Project status

This is a learning-focused extension. Future improvements can include validation, individual link deletion, tags, search, and export.
